> 가상의 가로선이 있다고 가정하고 한 층씩 물이 찰 수 있는 칸들을 확인한다.

1. 1 이상인 두 수 사이에 위치한 0 이하의 숫자 개수를 센다.
1. 모든 칸에서 1씩 뺀다.
1. 2차원 세계의 세로 길이만큼 반복한다.

> 예제 입력 1

```
3	0	1	4 → 1
2	-1	0	3 → 2
1	-2	-1	2 → 2
0	-3	-2	1 → 0
-1	-4	-3	0 → 0
```

> 예제 입력 2

```
3	1	2	3	4	1	1	2	→	0
2	0	1	2	3	0	0	1	→	3
1	-1	0	1	2	-1	-1	0	→	2
0	-2	-1	0	1	-2	-2	-1	→	0
-1	-3	-2	-1	0	-3	-3	-2	→	0
```
